# imam_shuttle_app

# وصف المشروع 
تطبيق Imam Shuttle هو نظام تقني متكامل خاص لطالبات جامعة الإمام محمد بن سعود الإسلامية لمساعدتهن في تنظيم وسهولة التنقل عبر منصة رقمية متطورة. 
يهدف التطبيق إلى تحسين كفاءة الوصول لخدمات النقل من خلال واجهات تفاعلية تضمن تجربة مستخدم آمنة وفعالة، مما يساهم في تلبية المتطلبات اليومية للطالبات بمرونة عالية


A new Flutter project.
